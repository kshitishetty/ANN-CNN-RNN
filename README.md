This project contains simple Python implementations of three important deep learning models:
ANN (Artificial Neural Network)
CNN (Convolutional Neural Network)
RNN (Recurrent Neural Network)
It’s designed for beginners to understand how these models work step by step.

What’s inside?
ANN: Basic forward pass, loss calculation, and weight update
CNN: Simple image convolution and kernel update
RNN: Sequence processing with hidden states
All examples are written using NumPy only (no complex libraries).

Purpose
This project is for learning and understanding the core concepts of deep learning in a simple way.
